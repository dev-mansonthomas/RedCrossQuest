Bienvenue dans RedCrossQuest ! Avant de commencer l'aventure, un peu de préparation !

Cher Administrateur/Administratrice,

Avant de pouvoir utiliser , il faut configurer l'application pour que votre Unité Locale puisse l'utiliser.
Ci-desssous, vous trouverez comment effectuer la configuration de l'application pour votre UL, cliquez sur le titre pour afficher/masquer les instructions.
Il est conséillé d'effectuer la configuration dans l'ordre présenté ci-dessous, mais ce n'est pas obligatoire.

  • Pourquoi : En créant plusieurs utilisateurs de l'application RCQ pour vos cadres quêtes, cela vous permet de finir plus rapidement le paramétrage initial de RCQ, et vous permet de distribuer les tâches (gérer préparation/départ/retour, comptage, suivi des chiffres). Note que pour afficher les graphiques "publique", vous ne pouvez le faire qu'avec un compte qui a le rôle "lecture seule".
  • Comment : Ajoutez un Quêteur
    1. Les utilisateurs de l'application RCQ sont vos cadres de quête qui vont faire les départs/retour des troncs, comptez les pièces etc...
    2. Un utilisateur de l'application RCQ est d'abord un quêteur, il faut donc d'abord créé le quêteur via "Ajouter un Quêteur"
    3. Une fois le quêteur créé, vous pourez cliquer sur "Faire de ce bénévole un utilisateur de l'application RCQ". Il recevera un email pour initialiser son mot de passe RCQ (le liens est valable une heure). Son login étant son NIVOL.
    4. Par défaut, son niveau d'accès est lecture seule. Voici les accès de chaque rôles :
      1. Lecture Seule: Liste des quêteurs et graphiques publiques
      2. Opérateur: rôle 'lecture seule' + Ajout/Mise à jour des quêteurs, préparation, départ, retour des troncs. Graphique Opérateurs.
      3. Compteur: rôle 'Opérateur' + Comptage des troncs et Graphiques Compteur
      4. Administrateur: rôle 'Opérateur' + Menu d'administration et graphique Administrateur
  • Pourquoi : Ces données vous permettront de comparer les chiffres journalier de cette année avec les précédentes années et ainsi de vous situer par rapport à votre objectif
  • Comment : Menu Administration->Résultats Journaliers (Avant RCQ)
    1. Choisissez l'année, puis cliquer sur Rechercher, un bouton apparait "Créez l'année XXXX"
    2. Pour chaque jour, créé le montant quêté ce jour la, puis cliquez sur "Sauvegarder"
    3. Répétez l'opération pour au moins les 3 dernières années (plus vous avez de données, mieux c'est)
    4. Si le détails par jour n'est pas retrouvable, prenez votre montant de l'année et répartissez le de façon linéaire grace à l'outils en bas de page
  • Pourquoi : A chaque tronc sera attribué un identifiant représenté sous la forme d'un QRCode , ce qui permet de fiabiliser les opérations de distrubtion des troncs et de comptages des pièces et billets
  • Comment : Menu Administration->Liste des Troncs
    1. Comptez vos troncs (boite métallique de collecte de pièces) que vous avez au local
    2. Indiquez ce chiffre dans le formulaire et cliquer sur "Créer de nouveau Troncs" (à Faire)
  • Pourquoi : La défnition des points de quête permet de déterminer où vos quêteurs vont quêter et dans quelle condition (pieton, feux, magasin, à combien).
    Vous pourrez visualiser sur une carte vos points de quête, lesquels donnent le plus, depuis combien de temps vos quêteurs sont sur le terrain.
  • Comment : Menu Administration->Point de Quête
    1. Nous avons automatiquement créé votre premier point de quête qui correspond à votre Local.
    2. L'adresse a été récupérée d'un référentiel Croix Rouge en 2016 et les coordonnées GPS calculé automatiquement à partir de l'adresse, aussi vérifiez l'exactitude de l'emplacement
    3. Vous pouvez déplacer le marqueur la carte pour afiner l'emplacement du point de quête. L'application vous proposera une nouvelle adresse que vous pouvez utiliser ou non.
  • Pourquoi : Chaque quêteur doit être identifié dans l'application, et ce afin de pouvoir lui envoyer des emails, de s'assurer qu'un mineur ne quête pas sur un point de quête non prévue pour les mineurs
  • Comment : Ajoutez un Quêteur
    1. Soit vous ajouter à la main tous vos quêteurs
    2. Vous construisez vous un fichier excel avec le format suivant
      1. Nivol
      2. Sexe: 0: femme, 1: homme
      3. Nom
      4. Prénom
      5. Date de naissance au format AAAA-MM-DD: Obligatoire pour les bénévoles mineurs, si inconnus pour les adultes, mettre 1922-12-22.
      6. Telephone mobile : au format +33XXXXXXXXX (pour vous permettre d'appeler le bénévole rapidement en cas de souci, l'application RCQ pourrait envoyer des SMS prochainement)
      7. Email (l'application RCQ envoie des emails)
      8. Secteur: 1: Action Sociale, 2: Secours, 4:Ancien Bénévole, inactif ou adhérent
    3. Et vous envoyez ce fichier à thomas.manson@croix-rouge.fr en précisant votre unité locale et votre mobile (en cas de souci). Je m'occuperais de charger vos quêteurs d'un coup.
